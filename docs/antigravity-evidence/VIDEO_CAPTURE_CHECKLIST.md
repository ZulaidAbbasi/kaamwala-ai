# Video Capture Checklist — KaamWala AI

> Use this checklist when recording both the Demo Video and Antigravity Usage Video.

---

## 🎬 Demo Video (4–6 min)

### Pre-Recording

```
[ ] App running on device/emulator
[ ] Internet stable (test a Places API call first)
[ ] Firestore cleared (fresh demo state)
[ ] Screen recorder configured (no system notifications)
[ ] Phone in Do Not Disturb mode
[ ] Audio input tested (clear narration)
[ ] Demo request copied to clipboard
[ ] System Diagnostics screen shows all green
```

### During Recording — 13 Steps

```
[ ] Step 1:  Show HomeScreen, narrate problem (30s)
[ ] Step 2:  Type/paste Roman Urdu request (20s)
[ ] Step 3:  Show AI understanding — BACKEND DECISION chip (25s)
[ ] Step 4:  Show provider list — REAL GOOGLE PLACES labels (25s)
[ ] Step 5:  Show REGISTERED vs ONBOARDING REQUIRED badges (20s)
[ ] Step 6:  Show ranking — 12-factor breakdown (25s)
[ ] Step 7:  Show price estimate — ESTIMATED chip (20s)
[ ] Step 8:  Show booking — FIRESTORE SAVED badge (20s)
[ ] Step 9:  Show notification preview — PREVIEW ONLY label (10s)
[ ] Step 10: Tap Provider Cancels — show recovery (30s)
[ ] Step 11: Show agent trace — scroll 7 entries (25s)
[ ] Step 12: Show baseline comparison — 9 dimensions (20s)
[ ] Step 13: Show outcome evaluation — score + grade (30s)
```

### Post-Recording

```
[ ] Video plays from start to finish
[ ] All 13 steps are clearly visible
[ ] No API keys visible in any frame
[ ] Audio is clear throughout
[ ] Duration: 4–6 minutes
[ ] All labels visible (SIMULATION, ESTIMATED, FIRESTORE SAVED, etc.)
```
