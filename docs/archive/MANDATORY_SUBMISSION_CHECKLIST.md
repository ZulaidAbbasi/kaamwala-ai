## 🔴 Mandatory Submissions

### 1. Mobile App Link

```
[ ] App runs on a real device or emulator
[ ] App link provided (Expo Go link, APK, or TestFlight)
[ ] App demonstrates the full demo flow end-to-end
[ ] No crashes during demo flow
[ ] All simulated actions labeled as "Simulation"
[ ] Urdu/Roman Urdu/English input accepted
```

**How to provide:**
- Option A: Expo Go published link (`exp://...`)
- Option B: APK file uploaded to Google Drive (public link)
- Option C: Screen recording if physical device sharing not possible
